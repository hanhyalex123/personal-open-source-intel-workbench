# 品牌头像与首页减法设计

## 目标
当前首页首屏虽然比之前更紧凑，但仍然存在信息堆叠、品牌区和首页正文重复、卡片数量过多的问题。新的目标是：把首页变成更干净的产品页头版，同时用一个本地可控的动漫女角色头像替换现有工具感徽章，提升识别度。

## 设计方向
- 品牌层：用本地 SVG 动漫女角色头像替换侧边栏与首页品牌区的文字徽章。
- 首页层：继续保留“重点结论 + 运行快照”这两个核心区块，但删除造成杂乱的碎卡与重复标签。
- 文档层：`README.md` 收敛成短版现状说明，只保留项目简介、能力、启动方式和截图。

## 首页调整
- 侧边栏品牌：头像 + 产品名 + 一行短描述。
- 首页头版：
  - 保留顶部栏目标签和产品名
  - 保留主标题与一句说明
  - 删除“值班视角 / 更新节奏 / 产品短宣言”碎卡
  - 删除冗余的重复摘要标签
  - 左侧聚焦“重点结论”主卡，右侧聚焦“运行快照”
- 下半区继续保留 `增量快讯` 与 `日报归档`

## 资源策略
- 不依赖外部图片源，新增本地 SVG 品牌头像资源，避免版权和稳定性问题。
- 头像风格为简化的动漫女角色侧脸/半身头像，控制在品牌徽章尺寸内，不做大型插画。

## 测试与验证
- 更新前端首页测试，去掉被删掉的文案断言，保留核心结构断言。
- 运行：
  - `npm test`
  - `npm run build`

# Homepage Visual Refresh Design

**Date:** 2026-03-15

## Goal

让「日报首页」在功能不变的前提下更像“编辑部头条”，突出日报主结论，并切换为冷静蓝灰的情报中心风格。

## User Intent

- 首页一眼能读出今天“最重要的结论”
- 报表信息弱化但仍易读（最近抓取、日报生成、调度）
- 排版更干净，视觉层级更清楚

## Constraints

- 不改变功能与数据结构
- 不修改后端 API
- 仍保留：日报卡、增量提醒、归档、统计信息

## Visual Direction

- 冷静蓝灰（低饱和、弱对比）
- 主结论为唯一视觉主角
- 统计信息以“轻量信息带”形式出现
- 背景使用冷色渐变 + 轻颗粒纹理

## Layout Plan (Single Column)

1. **编辑部头条区（Hero）**
   - 大标题：日报首页
   - 简短摘要：保留当前描述文案
   - 最新时间：最近抓取成功 / 最近日报生成

2. **轻量信息带**
   - 将原统计卡组收敛为一条横向信息带
   - 仍保留：结论总数、项目数、调度状态、最近日报生成、最近抓取成功

3. **日报卡主流**
   - 保持现有 ProjectSummaryCard
   - 增大卡片间距和内边距

4. **增量提醒 / 日报归档**
   - 维持现有顺序和内容
   - 标题层级稍降，避免与日报卡竞争

## Component Changes

- `IntelOverviewPage`
  - 结构上不新增功能，只调整层级与布局容器
  - 把统计区变成“轻量信息带”

- `index.css`
  - 全局背景切换为蓝灰渐变
  - 主色由绿/暖棕替换为冷蓝
  - 卡片与边框颜色调整为更冷静的灰蓝
  - 首页相关类（hero/stat-card）样式重排

## Testing Strategy

- 更新前端测试中首页相关文案与元素定位（不变功能）
- 运行 `npm test` 确保全量通过
